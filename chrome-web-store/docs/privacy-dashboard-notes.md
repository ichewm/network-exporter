# Chrome Web Store Privacy Dashboard Notes

Use these notes when completing the Chrome Web Store privacy practices form.

## Recommended Disclosure Framing

Network Exporter processes network request data locally inside Chrome DevTools. It does not transmit that data to the developer or any third party.

## Data Collection

If the dashboard asks whether the developer collects user data, the intended answer is:

- No data is collected by the developer.

Clarification:

- The extension locally displays and copies network data that Chrome DevTools exposes to the user.
- The developer does not receive, store, sell, or transmit that data.

## Single Purpose

Selecting, inspecting, and copying Chrome DevTools network request data as focused JSON or Markdown exports.

## Certification Statements

- The extension does not sell or transfer user data to third parties.
- The extension does not use user data for advertising.
- The extension does not use user data for credit-worthiness or lending.
- Humans do not read user data.
- The extension uses the minimum permissions necessary.

## Privacy Policy

Host `privacy-policy.md` on a public page before submission and use that URL in the dashboard.
