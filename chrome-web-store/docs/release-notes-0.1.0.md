# Release Notes 0.1.0

Initial Chrome Web Store submission package.

Included:

- DevTools panel named `Network Exporter`
- Request capture via `chrome.devtools.network`
- Request filtering
- Multi-select request export
- Single-request details
- JSON and Markdown clipboard exports
- Field selection with Defaults and Full presets
- Sensitive headers excluded by default
- Browser extension requests excluded by default
- Responsive layout for narrow DevTools panes
- Chrome Web Store icon and listing assets
