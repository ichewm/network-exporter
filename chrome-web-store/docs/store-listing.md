## Name

Network Exporter

## Summary

Select DevTools network requests and copy focused JSON or Markdown exports.

## Category

Developer Tools

## Language

English

## Detailed Description

Network Exporter adds a focused panel to Chrome DevTools for copying only the network requests and fields you need.

Instead of exporting a full HAR file or copying one request at a time, select a small set of requests, choose the fields you want, and copy clean JSON or Markdown directly to your clipboard.

Key features:

- DevTools panel for local network inspection
- Capture starts when DevTools opens, before the custom panel is selected
- Plugin-level Preserve log option for keeping requests across navigation
- Multi-select request table
- Request detail tabs for headers, payload, response, and metadata
- Filters for URL text, method, status, domain, MIME type, and resource type
- Error filtering for 4xx, 5xx, and failed requests
- Distinct row highlighting and counts for client errors, server errors, and failed requests
- Default exclusion for noisy browser extension requests
- Field-level export controls
- Defaults and Full field presets
- Collapsible export pane for narrow DevTools layouts
- Copy selected requests as JSON
- Copy selected requests as Markdown
- Sensitive headers excluded by default

Network Exporter is designed for debugging API workflows, preparing bug reports, sharing focused request examples, and pasting compact context into development tools.

Privacy and security:

- No remote servers
- No analytics
- No background traffic capture
- No host permissions
- No debugger permission
- Network data is processed locally inside the DevTools panel
- Data is copied only when you select requests and click a copy button

## Short Value Proposition

Copy only the network requests and fields you need from DevTools.
